----------------------------------------------------------

1.空Vue实例
----------
- 空Vue实例,也叫虚拟实例,大多做为中间过度实例使用
- 同级组件之间无法直接进行数据通信,但可通过第三方实现
- 这个第三方,可以通过创建一个空Vue实例代替

----------------------------------------------------------

2.实例作用域之间的切换
------------------
- 因为涉及到中间Vue实例,所以切换数据作用域很有必要
- 在通信过程中,要随时关注变量: this 的当前指向

----------------------------------------------------------

3.实例的mounted属性和$on函数
-------------------------
- mounted: 实例挂载点渲染完成后自动触发
- $on(事件名称, 事件回调函数)
- 用来监听实例上自定义事件,该事件必须由$emit触发
